之前没有系统学过python语言的语法，写的最多的语言是cpp，因此写下此博客来记录在实际coding中遇到的小问题

判断分支

不使用else if，而是使用elif关键字

真除法 & 整数除法

“/“表示普通除法，也叫真除法，返回浮点数结果（即使能整除）

“//“表示整数除法，返回整数结果，向下取整

浮点数运算取整

(1) 向下取整，用math.floor()，或者使用”//“运算符

(2) 向上取整，用math.ceil()

(3) 向零取整，用int(x)或math.trunc(x)

(4) 四舍五入，用round(x)

自加自减

“i++”或者”i–”这样的用法是非法操作，只能使用”i += 1”或者”i -= 1”

循环

一般使用for和while，可以加()，也可以不加

无穷大的表示

正无穷大用 float(‘inf’)，负无穷大用float(‘-inf’)，或者用math.inf(python 3.5+)，inf == math.inf

栈的使用

list[-1]等价于cpp中的stack.top()

list.append(x)等价于cpp中的stack.push()

list.pop()基本等价于cpp中的stack.pop() 差异在于cpp不返回弹出值，list返回弹出值

判断not list 或 len(list)==0即可实现栈的判空方法

队列的使用

deque是collections包中的双端队列，支持在两端进行O(1)时间复杂度的插入与删除操作

左侧入队 dq.appendleft(x)

右侧入队 dq.append(x)

左侧出队 dq.popleft(x)

右侧出队 dq.pop()

判断not dq 或 len(dq)==0即可实现判空

list的append()方法与拼接操作

list.append(x)是in-place修改list，而”l3=l1+l2”是通过创建新列表并赋值

append()方法没有返回值，”+”操作返回新list

append()方法只接收单个元素，”+”操作可以操作两个list

append()方法时间复杂度是O(1)的，”+”是O(n)

同时拼接操作还可以使用l1.extend(l2)方法，修改l1的值

求list中的最大最小值

内置方法min() max()，支持多种数据类型，且大小写敏感

对list进行排序

使用sorted(list)或者list.sort()方法，前者返回新列表，不改变原列表，后者直接原地修改列表进行排序。key参数控制排序规则，reverse参数控制排序方向

查找元素在列表中的位置

使用nums.index()方法

空值

python使用None来表示空值或不存在，类似cpp中的NULL或java中的null，判断是否为空是，用is，不用==

强制类型转换

int(x) 把浮点数和数字字符转为int，非数字字符不能直接转，需要ord()

float(x)

str(x) 数字转为数字字符，浮点数也可以转

bool(x)

list(x) 转str比较常用

set(x) list转为集合（可以去重）

字符和整数类型之间的转换

通过ord() 和chr()方法来实现，ord()实现从字符到整数的转换，chr()则相反

字符串与列表的类型转换

出于安全性的考虑和对哈希的支持，python中的str类型变量一旦创建就不能被修改

str类型变量的拼接、切片等操作实际上是创建了新的str变量

如果需要频繁修改字符串，可以让str与list互相转换

dict&set

dict字典是键值对的集合，类似于cpp中的map。字典具有无序性，键是唯一的，重复会覆盖。set是无序、不重复的元素的集合，类似于cpp中的set。元素唯一，自动去重，不支持索引

使用小顶堆，大顶堆

import heapq

默认是小顶堆，想要实现大顶堆，可以通过对数值取反
